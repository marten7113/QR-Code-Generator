# QR Generator 

I made a **QR Code Generator** using Python.

## Prerequisites
Before you run the script, you need to install the `qrcode` and `Pillow` libraries.

## Installation
Run the following commands in your terminal:

```bash
pip install qrcode
pip install Pillow
```

## How to use
1. Run the script.
2. Enter the URL you want to convert.
3. Save your image!
